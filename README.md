DIRECTORY STRUCTURE
-------------------

```
/
    app/                应用(想结合 django)
    bin/                应用脚本
    doc/                文档
    cache_files/        csv
    common/             公用
    config/             配置
    README.md           readme.md
    venv/               虚拟环境
```

## 虚拟环境

```python
//激活虚拟环境
$ python3 -m venv venv(环境名称)
$ source venv/bin/activate

// 退出虚拟环境
$ deactivate
```
